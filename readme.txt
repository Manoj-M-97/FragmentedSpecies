FRAGMENTED SPECIES

Description:
The implementation is done using HTML, CSS, Javascript, PHP, MySQL.
This website is an exhibitional website which is based on the rare, endangered, critically endangered, Vulnerable and extinct animals.
The picture of the animals are done using polygons.
It has videos based on the animals embedded in. It also has links to Facebook, Instagram and Twitter so that these can be shared there.
It includes the information of threats to and treatments for all the animals.
It has a link to World Wildlife Fund, and links to travel destinations which is suggested based on the habitat of the animal.
There is also a chatbot which answers questions related to the website.

About the Repository:
Different features which are implemented are in different files which are named accordingly.
The folder named 'Assets' has all the media from which it has to be accessed.
